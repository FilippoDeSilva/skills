9. **Docker**: Docker skills for containerization.

## Agent installation for agentic development

```bash
npx skills add https://github.com/FilippoDeSilva/skills --skill <skill-name>
```

## Cloned installations (Linux/Mac)

```bash
git clone https://github.com/FilippoDeSilva/skills
cd skills
chmod +x install.sh
./install.sh
```
